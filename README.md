# Descripción del Proyecto

Este trabajo práctico consiste en el desarrollo de un sistema básico de análisis de ventas utilizando Python. 

El proyecto integra herramientas de gestión colaborativa y control de versiones mediante Jira y GitHub, simulando un entorno de trabajo profesional basado en metodologías ágiles.

# Escenario Seleccionado

Escenario B – Análisis de Ventas de una Pequeña Empresa.

El sistema procesa información de ventas y genera indicadores automáticos junto con una visualización gráfica de resultados.
